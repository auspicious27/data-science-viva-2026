# Module 1: Introduction to Data Science, Machine Learning & AI

## Individual Topic Explanation

### Data Science in business terms
Business problem ko data ke help se solve karna. Example: sales kam kyun ho rahi hai, customer churn kaise reduce hoga.

### Machine Learning in business terms
Historical data se pattern seekh kar future prediction ya decision support dena.

### AI in business terms
Machines ko intelligent behavior dena, jaise chatbot, image detection, recommendation.

### Data Science vs Data Analytics vs ML vs AI
Analytics past data explain karta hai, Data Science insight plus model banata hai, ML prediction seekhta hai, AI intelligent system banata hai.

### Data Scientist role
Business problem samajhna, data clean karna, analysis/model banana, insights communicate karna.

### ML Engineer role
Model ko production-ready banana, deployment, monitoring, optimization aur scalable pipeline banana.

### Data Science lifecycle
Business Problem -> Data -> Analysis -> Model -> Insight -> Decision.

### Structured data
Rows-columns format, jaise Excel, SQL table.

### Semi-structured data
Partial structure hota hai, jaise JSON, XML, logs.

### Unstructured data
Fixed table format nahi hota, jaise images, videos, PDFs, audio, text.

### Industries using DS/ML/AI

### Data-driven decisions
Gut feeling ke bajay evidence aur patterns ke basis par decisions lena.

### Project: Business Problem Framing
Real business issue choose karke usko measurable DS/ML problem statement mein convert karna.

### Business objective
Business ko kya improve karna hai, jaise churn reduce, fraud catch, sales forecast.

### Required data
Problem solve karne ke liye kaunsa data chahiye, jaise transactions, customer details, dates, labels.

### Expected analytical/ML output
Final result kya hoga: report, prediction, recommendation, risk score ya segment.
